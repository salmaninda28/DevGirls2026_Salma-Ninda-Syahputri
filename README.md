# DevGirls2026_Salma-Ninda-Syahputri

Study tracker adalah platform yang memudahkan  manajemen dan produktivitas mahasiswa dalam menginput tugas-tugas kuliah. website ini merupakan sentralisasi tugas, di mana mahasiswa bisa mengumpulkan daftar tugas dalam satu  tempat untuk meastikan tidak ada deadline yang terlewat. 

Proyek ini dibangun dengan mengintegrasikan 3 teknologi dasar, yaitu: HTML, CSS, dan Java Script. Saya menggunakan html untuk membangun struktur dasar dari website ini sendiri. sedangkan untuk membuat tampilan sedemikian rupa Saya menggunakan css dengan mengatur tata letak, warna, dan font agar lebih menarik. Java Script digunakan untuk memberikan logika fungsional dan membuat website lebih dinamis dan fungsional.

Website ini memiliki beberapa fitur, yaitu: mahasiswa dapat melakukan pencarian tugas berdasarkan kategori mata kuliah dan bisa melihat status progres tugas tersebut apakah sudah selesai atau belum. Dengan  meng-klik tugas itu sendiri, mahasiswa bisa melihat detail tugas yang ada  atau menghapusnya jika sudah selesai. Selain itu, ada dashboard untuk memantau jumlah tugas dengan deadline hari ini, tugas yang sudah selesai dan belum selesai. Setelah itu, mahasiswa bisa menambahkan tugas di task entry. fitur-fitur itu merupakan fitur wajib yang akan memudahkan  mahasiswa dalam mengatur pengelolaan tugas.
