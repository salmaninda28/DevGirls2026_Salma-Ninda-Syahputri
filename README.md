# DevGirls2026_Salma-Ninda-Syahputri

website ini mempresentasikan sebuah study tracker yang memudahkan  manajemen dan produktivitas mahasiswa dalam menginput tugas-tugas kuliah. website ini memiliki beberapa fitur, yaitu: task entry, tambah tugas, dashboard untuk menunjukkan tugas dengan deadline hariini, jumlah tugas yang sudah selesai dan belum selesai. fitu-fitur itu merupakan fitur wajib yang akan memudahkan  maahsiswa dalam mengatur pengelolaan tugas.

aku menggunakan html untuk membuat struktur dasar dari website ini sendiri. sedangkan untuk membuat tampilannya sedemikian rupa aku menggunakan css dengan mengatur tataletak, warna, dan font agar lebih menarik. Java Script digunakan utnuk membuat website lebih dinamis dan fungsional.
